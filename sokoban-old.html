Terminal Sokoban

is a recreation of Sokoban for a university assessment where the goal was to make a text based game.

Github

Project Source

Inspiration behind the idea

The idea for this project came from merging two ideas. The first being creating a rendering system using ANSI escape codes in the terminal. I had this idea a while before and was wondering if it was possible. The introduction of an assessment for creating something in the terminal reintroduced this idea in my mind. The second idea being recreating Sokoban, I saw the game a while back and thought the puzzles that were created through it were interesting and thought provoking.
